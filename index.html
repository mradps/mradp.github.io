KERETA EKONOMI
Kereta api ekonomi adalah kereta api penumpang yang kelas nya di bawah kelas bisnis.
kereta api ekonomi sekarang ini sudah dilengkapi dengan AC dengan model AC split.
Kereta ekonomi ini berkapasitas hingga 106 penumpang dengan formasi tempat duduk 3-2.
kereta ekonomi menjadi idaman dan primadona banyak orang dikarenakan harnya yang terjangkau.
meskipun sering kalah bersilang dengan kereta api eksekutif, peminat dari kereta ini tidak berkurang.
Geta Production menyediakan adddons kereta ekonomi tersebut
dengan kulitas dan detail yang serupa dengan kereta aslinya.
FEATURE :
Real scale & HD
Animated door
Pasang / lepas rantai
Semboyan 21 siang
Semboyan 21 malam
Nomor Urut rangkaian
Include 7 macam livery
Include 40 macam nama KA
Warna atap yang bisa dirubah
Warna window Frame yang bisa diubah

Trainz Simulator Android

Trainz Simulator 2010
